Для отладки программ будем использовать отладчик Turbo Debugger (TD.exe).

Запуск:
	td PROG.EXE

Интефрейс:
По-умолчанию открыты 4 главных окна:
	CPU: окно с текущим указателем инструкций (текущий адрес CS:IP, по которому отображаются данные как декодированные ассемблерные инструкции).
	Registers: окно с регистрами CPU.
	Dump: окно с данными (Отображение памяти по указанному адресу SEGMENT:OFFSET).
	Stack: окно с данными по текущему положению указателя стека (SS:SP).
Другие окна можно открыть в меню View.

Управление:
	Мышью можно выбирать активное окно.
	В окне Dump можно вводить просматриваемый адрес: Ctrl+G.
	С помощью ПКМ в оке Dump можно открыть контекстное меню, в котором можно выбрать вид отображеня данных, перейти по указываемому адресу и прочее. Аналогичное контекстное меню есть и в окне CPU.
	В окне Registers можно с помощью контекстного меню менять значения регистров процессора.

Трассировка кода:
	Шаг с обходом: F7
	Шаг с заходом: F8
	Запуск программы до прерывания: F9
	Поставить точку останова (Breakpoint) при наведении и выборе текущей строки в отображении окна CPU: F2.
	Поставить точку останова по указываемому адресу: Alt+F2.
