СИНТАКИС ОБЪЯВЛЕНИЯ МЕТКИ:
	ИМЯ_МЕТКИ :

	label_name: mov ax, bx
	label_name1:
		mov ax, bx
		
Правила для объявления меток: метка должна начинаться с буквы и может сожержать цифры и символ '_', при этом все метки должны быть уникальными и не повторять ключевые слова компилятора.


РЕЖИМЫ АДРЕСАЦИИ ПЕРЕХОДОВ.	
Режимов адресации переходов называется способ нахождения цели перехода, т. е. Адреса, куда передается управление.

    1. Внутрисегментный короткий РА — адрес перехода равен сумме счетчика команд и однобайтового смещения. (IP + disp [8] = АП)
    2. Внутрисегментный прямой РА — адрес цели расположен в коде команды. (ЕА → адрес перехода)
    3. Внутрисегментный косвенный РА — адрес записан в регистре или ячейке памяти, адресуемой с помощью любого РА данных. (reg→EA→адрес перехода \ ЯП→EA→адрес перехода).
    4. Межсегментный прямой РА — длинный указатель (сегмент 2Б и смещение 2Б) располагается в коде команды. (КОП IPnew CSnew)
    5. Межсегментный косвенный РА — длинный указатель располагается в двух соседних словах (2Б \ WORD) памяти, младший из которых адресован любым РА данных. ( КОП [ЯП] )
	
	
ИНСТРУКЦИИ СРАВНЕНИЯ СMP И TEST.

Команда CMP предназначена для сравнения чисел. Она выполняется аналогично команде SUB: из первого операнда вычитается второй, но результат не записывается на место первого операнда, изменяются только значения флагов.

    cmp al, 5     ; Сравнение AL и 5
    jl label     ; Переход, если AL < 5 (числа со знаком)

    cmp al, 5     ; Сравнение AL и 5
    jb label     ; Переход, если AL < 5 (числа без знака)
	
Команда TEST работает аналогично команде AND, но также результат не сохраняется, изменяются только флаги. 

	test bl, 00000100b 		; Проверить состояние 2-го бита BL
    jz label             	; Переход, если 2-й бит равен 0

	
ИНСТРУКЦИИ ПЕРЕХОДОВ.
Безусловный переход — это переход, который выполняется всегда. 
Безусловный переход осуществляется с помощью команды JMP. У этой команды один операнд, который может быть непосредственным адресом (меткой), регистром или ячейкой памяти, содержащей адрес.

	jmp label    		; Переход на метку
	...
    jmp bx       		; Переход по адресу в BX
	...
    jmp word ptr [bx] 	; Переход по адресу, содержащемуся в памяти по адресу в BX
	
Условные переходы
Условный переход осуществляется, если выполняется определённое условие, заданное флагами процессора (кроме одной команды, которая проверяет CX на равенство нулю).
Существует много команд для различных условных переходов. Также для некоторых команд есть синонимы (например, JZ и JE — это одно и то же).

! СМ. ТАБЛИЦУ condition_jmp_instr_table.jpg

Некоторые команды применяются для беззнаковых чисел, а другие — для чисел со знаком. Сравнения «выше» и «ниже» относятся к беззнаковым числам, а «больше» и «меньше» — к числам со знаком.

При компиляции код инструкций переходов выглядит следующим образом: КОД OFFSET	(например код инструкции перехода jl label выглядит следующим образом: 7C04h), где OFFSET – смещение относительно текущей инструкции (текущего значения IP) до места назначения ПЛЮС длина инструкции перехода. Например:

	...
ip=0x12:  7С04	jl 0x18		
	...
ip=0x18:	48	dec ax
	...

В данном примере КОД инструкции условного перехода JL равен 0x7С, OFFSET до места назначения равно 0x04, посчитав по формуле 
IP_DEST = OFFSET + IP + INSTR_SIZE
IP_DEST = 0x04 + 0x12 + 2	= 0x18

Инструкции переходов, при котором значение OFFSET лежит в диапазоне знакового байта (-128 до 127) называются «КОРОТКИМИ» (SHORT), а с диапазоном знакового слова (-32768  до 32767) «БЛИЖНИЙ» (NEAR) (в переделах текущего сегмента — он же внутрисегментный ближний).


ИНСТРУКЦИИ ЦИКЛОВ.

Для организации цикла предназначена команда LOOP. У этой команды один операнд — имя метки, на которую осуществляется переход. В качестве счётчика цикла используется регистр CX. Команда LOOP выполняет декремент CX, а затем проверяет его значение. Если содержимое CX не равно нулю, то осуществляется переход на метку, иначе управление переходит к следующей инструкции. 
Содержимое CX интерпретируется командой как число без знака. 
Метка должна находиться в диапазоне -127 ... +128 байт от команды LOOP (короткий переход).

	mov cx, 3
	mov ax, 0
	
	label:
		inc ax
		loop label	; cx--; если сx != 0 переход к метке label, иначе переход к следующей инструкции
	...
	
Кроме команды LOOP и команд условных переходов существуют ещё две команды, позволяющие организовывать циклы. Это команды LOOPZ (или её синоним LOOPE) и LOOPNZ (синоним — LOOPNE).
Действие этих команд аналогично LOOP, за исключением того, что дополнительно анализируется флаг нуля ZF. 
Переход к метке цикла осуществляется в том случае, если после декремента содержимое CX не равно 0 и выполняется условие: ZF=1 (для команды LOOPZ/LOOPE) или ZF=0 (LOOPNZ/LOOPNE).

	mov cx, 5
	mov ax, 3
	
	label:
		dec ax
		loopnz label	; cx--; если сx != 0 И ZF == 1 переход к метке label, иначе переход к следующей инструкции
	...
